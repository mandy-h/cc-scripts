# userscripts
## ce-keyboard-gamepad-controls
Adds keyboard and gamepad functionality to the Click Exchange.
## click-and-drop-ui-modifications
Modifies the click-and-drop UI used in quick_folder_sort.php, the TS, etc.
## collection-comparison-filter
Adds a form on the collection comparison page to allow advanced filtering.
## cooldown-timer
Adds cooldown timers to games.
## freeze-animated-adopts
Freezes all user-owned adoptable images.
## item-page-adopt-search
Adds search options to the dropdown adoptable selector on item pages.
## mobile-friendly-ce
Modifies CE page styling for a better mobile experience.
## shop-mp-ts-highlighting
Highlights missing adopts and army adopts in the shops, marketplace, and trade station.
## show-folder-id
Changes Edit links to "Edit {folder ID}" on the "Manage Folders" page.
## tribal-tracking-fight-phase-hotkeys
Adds keyboard controls to Tribal Tracking fight phase.
## tribal-tracking-gamepad
Adds an on-screen gamepad and keyboard controls to Tribal Tracking.
