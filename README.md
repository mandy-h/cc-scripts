# cc-scripts
A collection of userscripts and console scripts for adding small enhancements on ClickCritters.
* Scripts in the `console-scripts` folder are run directly in the browser console. Scripts in the `userscripts` folder are run using a userscript manager extension such as Tampermonkey, Violentmonkey, Greasemonkey.
* To install a userscript, navigate to the userscript file in Github and click the "Raw" button located above the code viewer.
* For scripts that have specific instructions or notes, there will be a "how to use" section either near the top of the file or right after the script metadata (starts and ends with `// ==/UserScript==`).
* All scripts were tested in Firefox. Scripts may break in the future due to site updates.
